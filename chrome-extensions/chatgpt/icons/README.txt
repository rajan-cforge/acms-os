Icon Placeholders

For this MVP, you need to create PNG icons in these sizes:
- icon16.png (16x16 pixels)
- icon48.png (48px48 pixels)
- icon128.png (128x128 pixels)

Quick way to create icons:
1. Use any image editor (Photoshop, GIMP, Canva, etc.)
2. Create a simple logo with "ACMS" text or brain icon
3. Export as PNG in the sizes above
4. Place in this directory

Or use online tools:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

For now, the extension will work without icons (Chrome shows default icon).
